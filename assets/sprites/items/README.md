# Item Sprites

This directory will contain item and loot sprites.

## Categories:
- Weapons (swords, bows, staffs)
- Armor (helmets, chest plates, boots)
- Potions (health, mana, buffs)
- Collectibles (coins, gems, keys)
- Consumables (food, scrolls)

## Recommended format:
- PNG files with transparency
- 16x16 or 32x32 pixels
- Consistent art style
