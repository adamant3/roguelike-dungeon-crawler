# Tile Sprites

This directory will contain tileset textures for building dungeon environments.

## Tileset Types:
- Floor tiles (stone, wood, dirt)
- Wall tiles (brick, stone, wood)
- Decorations (cracks, moss, blood stains)
- Interactive objects (doors, chests, switches)

## Recommended format:
- PNG files with transparency where needed
- Tileset images organized in grids
- Power-of-2 dimensions
- Consistent tile size (e.g., 16x16 or 32x32)

## Usage:
Import tilesets into Godot and configure them in TileMap nodes.
