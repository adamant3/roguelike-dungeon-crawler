# Player Sprites

This directory will contain player character sprites.

## Recommended format:
- PNG files with transparency
- Power-of-2 dimensions (e.g., 16x16, 32x32, 64x64)
- Pixel art style

## Naming convention:
- `player_idle.png` - Standing still animation
- `player_walk.png` - Walking animation sprite sheet
- `player_attack.png` - Attack animation
- etc.
