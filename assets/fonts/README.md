# Fonts

This directory contains font files for the game's UI and text.

## Recommended fonts for pixel art games:
- Pixel fonts with consistent character width
- TTF or OTF format
- Consider free/open source options from:
  - Google Fonts
  - itch.io
  - opengameart.org

## Usage in Godot:
1. Import font file
2. Create a FontFile resource
3. Assign to Label or RichTextLabel nodes
4. For pixel-perfect text, disable anti-aliasing

## Licensing:
Make sure any fonts you use have appropriate licenses for game distribution.
